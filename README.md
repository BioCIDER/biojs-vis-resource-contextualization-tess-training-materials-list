## Getting Started
Install the module with: `npm install biojs-vis-events`

```javascript
var events = require('biojs-vis-events');
events.hello("biojs"); // "hello biojs"
```

## Documentation

#### .hello(name)

**Parameter**: `name`
**Type**: `String`
**Example**: `biojs`

The 'hello' method is responsible for showing a name.

How to use this method

```javascript
events.hello('biojs'); // "hello biojs"
```
